# Changelog - Music Sheet Viewer V5.0

## v5.0.0 - 2026-03-08
- 플레이리스트 온라인 동기화 (Google Sheets 기반 업로드/다운로드)
- 가사 DB 경로를 상대경로로 저장 (악보 폴더 변경 시에도 가사 유지)
- 가사 DB 경로 매칭 오류 수정 (슬래시/대소문자 정규화)
- 환경설정에 스프레드시트 ID 설정 및 서비스 계정 이메일 복사 기능 추가
- 조와 가사 영역 기본 펼침으로 변경
- 리스트 항목 간격 축소

## v1.2.0 - 2026-03-08
- 텍스트 슬라이드 플레이리스트 저장/불러오기 지원
- 텍스트 슬라이드의 폰트 종류, 크기, 테마 등 모든 설정이 저장됨
- 플레이리스트 미리보기에 텍스트 슬라이드 표시

## v1.1.0 - 2026-03-08
- 대규모 업데이트 및 아카이브 정리
- 모니터 감지 기능
- 정렬 문제 해결
- 디자인 변경

## v1.0.0
- 트리에서 삭제 기능 추가
- DB 버튼 크기 조정
- 곡정보 UI 개선
- 인터미션 배경, 로고 화면, 블랙스크린 화면 개선
- 악보 데이터베이스 연동 설정
- GUI / UI 개선
- 초기 릴리즈
